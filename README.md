# AI Job Analyzer System

AI Job Analyzer System is a Flask-based web application that analyzes resumes and recommends jobs based on extracted skills.

## Features
- Resume upload
- Skill extraction from PDF
- Job recommendation
- Match score
- Simple UI

## Tech Stack
- Python
- Flask
- Pandas
- PyPDF2
- HTML
- CSS
- Bootstrap

## How to Run

```bash
pip install flask pandas PyPDF2
python app.py
